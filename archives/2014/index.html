GRUB2 Rescue 模式时使用的开机命令

Global CSS

由于各浏览器都有自身默认的样式，且不同浏览器的默认样式会有些不同。为了方便起见，在写CSS时，首先会使用CSS Reset清除浏览器的默认样式，以避免浏览器默认样式的干扰。而网页的整体风格的样式，比如：将页面中的a元素全部不加下划线（即：a{text-decoration: none;}），也会在CSS文件的开头。因而可以将CSS Reset修改并加上网页整体风格的样式，构成Global CSS。这份Global CSS将是网站中直接使用的CSS，一般放在整个CSS文件开头，或单独作为一个CSS文件引入。
