GRUB2 Rescue 模式时使用的开机命令

使用 Grub2 作为开机选单时，若设置错误或安装失败，而在开机时不显示启动选单，而是显示 grub-rescue> 这一提示符。那就需要使用者手动输入开机命令来启动操作系统。
或者 Grub2 选单正常时，也可以按照屏幕下方的提示，按下 c 来进入命令列模式，此时可以看到 grub> 提示符。
